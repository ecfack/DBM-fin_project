#### 簡介：
這是一個關於各家餐廳的討論平台，
可以藉由登入帳號，來編輯自己的店家資訊或發文評論店家。

#### 執行方法：
1. copy這份專案
2. 使用xampp打開Apache和MySQL
3. 建立DB叫foodmap，把foodmap.sql載入
4. 由localhost移動到此專案所在的路由下的Homepage.php，
   ex：http://localhost/DBM_fin_project-PHP/Homepage.php
4. 開始使用

#### 使用說明
當作普通網站使用就好，不過新增店家資料目前只能手動打進DB
